# API & SDK Reference

Developer reference for integrating with Lumina Engine's APIs and SDKs.

## Overview

The Lumina API allows developers to:

- Submit encrypted intents on behalf of users
- Query indicative pricing and depth (non-revealing)
- Retrieve post-trade receipts and inclusion proofs
- Initiate private send and payment request flows
- Monitor settlement status and shadow ledger state
